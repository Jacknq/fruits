# fruits
Vue 2 typescript example with vuetify, vuex modules and fetch (mande)
## Project setup
```
npm install
```

### Runs api - second terminal
```
npm run api
```

### Compiles and hot-reloads for development
```
npm run serve
```


### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```
